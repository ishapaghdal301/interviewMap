## Installation

1. Clone the Repository

```bash
  git clone https://github.com/ishapaghdal301/InterviewMap.git
  cd InterviewMap
```

2. Backend Setup 
```bash
  cd backend
  npm install
  node server.js
```

3. Frontend Setup
```bash
  cd frontend
  npm install
  npm start
```
